# 🚀 OpenCV Haar Cascade Projects

A collection of Computer Vision projects developed using Python, OpenCV, Haar Cascade Classifiers, and Streamlit for real-time face, eye, and car detection applications.

---

## 📌 Features

- Face Detection from Images
- Eye Detection using Haar Cascade
- Real-Time Face & Eye Detection using Webcam
- Car Detection from Video
- Streamlit Web Application
- OpenCV Image & Video Processing

---

## 🛠️ Technologies Used

- Python
- OpenCV
- NumPy
- Streamlit
- Haar Cascade Classifiers

---

## 📂 Project Files

| File Name | Description |
|------------|-------------|
| `face_detection.py` | Detects faces from images |
| `face_eye_detection.py` | Detects faces and eyes from images |
| `face_eye_usingwebcam.py` | Real-time face & eye detection using webcam |
| `car.py` | Detects cars from video |
| `streamlitFaceeye.py` | Streamlit web app for face & eye detection |

---

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/opencv-haar-cascade-projects.git
```

### Move to Project Folder

```bash
cd opencv-haar-cascade-projects
```

### Install Required Libraries

```bash
pip install opencv-python numpy streamlit pillow
```

---

## ▶️ Run the Projects

### Face Detection

```bash
python face_detection.py
```

### Face & Eye Detection

```bash
python face_eye_detection.py
```

### Webcam Detection

```bash
python face_eye_usingwebcam.py
```

### Car Detection

```bash
python car.py
```

### Streamlit Web App

```bash
streamlit run streamlitFaceeye.py
```

---

## 📁 Required XML Files

Download these Haar Cascade XML files and place them inside the project folder:

- `haarcascade_frontalface_default.xml`
- `haarcascade_eye.xml`
- `cars.xml`
